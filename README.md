# 游戏规则
- *堆叠牌堆*按照从上到下从大到小依次递减，且红黑花色交替的方式进行堆叠
- *空位*只能放置四种花色的 K 
- 右上方的*已完成牌堆*分别摆放四种花色的牌，其中 A 可以直接放上去，而其他的牌只有比其牌面数值小 1 且花色完全相同的牌在上面时才能放上去
- 下方有些牌背面朝上放在其他亮出的牌的下方，称为*隐藏牌堆*。它们只有在其上方的牌移走后才能露出
- 左上方的*随机牌堆*中摆放着可供玩家直接操作的纸牌
